# mentalbert-historical-mental-health-discourse

# Investigating Word Associations in Historical Mental Health Discussions (MentalBERT)

This repository contains the academic poster for a project in the course "Embedded Meaning: Word Embeddings in Theory and Practice" in the MA Data and Discourse Studies (TU Darmstadt) exploring how a modern mental health–specific transformer model (MentalBERT) captures word associations in historical mental health discourse.

## Overview
The project uses a historical corpus of digitized newspaper pages (1858–1963) from the Library of Congress *Chronicling America* collection consists of 188 digitized newspaper articles/pages linked to mental health topics and terminology. The project analyzes semantic proximity between mental-health-related terms via embedding similarity. The goal is to see which associations appear strong, and where a contemporary model’s “understanding” may diverge from historical usage.

## Methods (high level)
- MentalBERT embeddings (Hugging Face Transformers)
- Cosine similarity to identify top associations for key terms
- Visualizations: similarity heatmaps, t-SNE projection, and a network graph of term associations
